# Social Media Images for ZFish

This directory contains the social media images used for SEO and link previews.

## Required Images

### 1. og-image.png (1200x630)
- Used for Open Graph (Facebook, LinkedIn, etc.)
- Location: `/public/og-image.png`
- Dimensions: 1200x630 pixels (1.91:1 aspect ratio)
- Shows terminal/command line aesthetic with ZFish branding

### 2. twitter-image.png (1200x600)
- Used for Twitter/X card previews
- Location: `/public/twitter-image.png`
- Dimensions: 1200x600 pixels (2:1 aspect ratio)
- Optimized for Twitter's card display

## How to Generate Images

### Option 1: Use the HTML Templates
1. Open `og-image.html` in a browser
2. Set browser window to 1200x630 pixels
3. Take a screenshot or use browser dev tools to save as PNG

### Option 2: Use Online Tools
- [Canva](https://www.canva.com/) - Design custom images
- [Social Image Generator](https://social-image-generator.vercel.app/) - Code-based generation
- [Open Graph Image Generator](https://og-image.vercel.app/) - Vercel OG Image

### Option 3: Use Design Software
- Figma, Adobe Photoshop, or GIMP
- Use the terminal/command line theme
